Crocheting

Crocheting is using a singular hook on yarn. You can create stuff animals, clothing or anything basically made from yarn.

When using a hook there could be many sizes. You could see more here.

Hook Type

There are different kinds of Hooks used for crochet. There could be plastic, steel, and aluminum. The different hooks depend on the person. If you want a more comfortable hook, prefer one with rubber handle to have a better grip on it. Wood will have a smooth crocheting with certain type of yarn.

Hook Style

There can be different styles. Like inline (with a flat, uniform throat) or tapered (with a rounded, sloped throat). As seen below:

Some Tips

  • A tip to finding a good hook for certain yarn. The size of the yarn has to be the same size of the hook. This will garantee no holes or gaps in your project and have a clean looking project.

  • Depending on the yarn you use, you can use different hook sizes on the yarn to have a bigger project. If you were to use a yarn 4 Size, Hooks 5 mm, 3.5 mm, and 2.75 mm will work on this yarn but the finish project will have different sizes because the yarn would have more space apart when the hook size is bigger.
